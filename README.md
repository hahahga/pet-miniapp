# Pet Management WeChat Mini Program

## Project Overview
A comprehensive WeChat Mini Program for pet management, featuring pet social networking, health tracking, service booking, and more.

## Features
- **Pet Profile Management**: Add and manage pet information
- **Health Records**: Track pet health and vaccination records
- **Social Features**: Share pet moments and connect with other pet owners
- **Service Booking**: Book pet services (grooming, veterinary, etc.)
- **Location Services**: Find nearby pet services
- **Wallet System**: Manage payments and transactions

## Project Structure
```
├── frontend/         # WeChat Mini Program frontend
│   ├── pages/        # Mini Program pages
│   ├── components/   # Reusable components
│   └── assets/       # Images and icons
└── backend/          # Backend services
    ├── Address Book Backend/
    ├── Core Concepts/
    └── me/
```

## Key Technologies
- WeChat Mini Program Framework
- Python Flask/FastAPI Backend
- SQLite Database
- RESTful API

## Frontend Pages
- Home: Pet feed and activities
- Health: Health records and tracking
- Social: Pet community
- Service: Service booking
- Me: User profile and settings

## Requirements
- WeChat Developer Tools
- Python 3.8+
- Flask
- SQLite

## Usage
1. Import frontend folder to WeChat Developer Tools
2. Run backend server: `python app.py`
3. Configure API endpoints in frontend
